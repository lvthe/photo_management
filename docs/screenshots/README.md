# Ảnh màn hình

Đặt ảnh chụp màn hình vào thư mục này đúng tên file để README ở trang chủ tự hiển thị:

| Tên file | Màn hình |
|----------|----------|
| `dashboard.png` | Trang chủ (lưới chức năng) |
| `order.png` | Quản lý đơn in |
| `search.png` | Tìm kiếm / lọc đơn |
| `customer.png` | Quản lý khách hàng |
| `service.png` | Quản lý dịch vụ in |
| `dark_mode.png` | Giao diện tối (tuỳ chọn) |

**Mẹo chụp:** trên emulator/thiết bị, ảnh dạng `.png` dọc (khoảng 1080×2340) là đẹp. Có thể chụp bằng Android Studio (Logcat ▸ Screenshot) hoặc nút chụp màn hình của máy.
